    HOW DO THEY WORK BEHIND THE SCENES

  • Facebook has a system known as Gatekeeper that allows them to run different code for different groups of users (it basically introduces different conditions in the code base).
  • This enables Facebook to roll out new features gradually, conduct A/B testing, activate certain features only for Facebook employees, and so on.
  • Gatekeeper also allows Facebook to perform "dark launches," which involve activating elements of a feature behind the scenes before it goes live.
